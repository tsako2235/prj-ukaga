# 06. 管理画面

通常のウィンドウ(フレームあり・リサイズ可)。トレイアイコンのメニュー、またはキャラの右クリックメニューから開く。React + TypeScript で実装。

## 画面構成(サイドバー + コンテンツ)

### 1. LLM設定

| 項目 | UI | 備考 |
|---|---|---|
| プロバイダ | セレクト(Ollama / OpenAI互換) | |
| 接続先URL | テキスト | プロバイダ切替でデフォルト値を自動入力 |
| APIキー | パスワード入力 | OpenAI互換のみ表示 |
| モデル | セレクト | 「接続テスト」成功時に `llm:listModels` で動的取得 |
| Temperature | スライダー(0〜2) | |
| 履歴送信数 | 数値入力 | contextLimit |
| 接続テスト | ボタン | healthCheck結果と失敗理由を表示(例: 「Ollamaが起動していません」) |

### 2. 音声設定

| 項目 | UI | 備考 |
|---|---|---|
| 音声を使う | トグル | OFFでもテキスト会話は動く |
| エンジン | セレクト(VOICEVOX / AivisSpeech / COEIROINK / カスタム) | 切替でデフォルトURL自動入力 |
| 接続先URL | テキスト | |
| 話者 | セレクト(キャラ名 / スタイル) | `voice:listSpeakers` で動的取得 |
| 話速 / 音高 / 音量 | スライダー | |
| テスト再生 | ボタン | 固定フレーズを合成して再生 |

### 3. キャラクター設定

| 項目 | UI | 備考 |
|---|---|---|
| 名前 | テキスト | システムプロンプトの `{name}` に反映 |
| Live2Dモデル | ファイル選択 | `.model3.json` を指定。選択後マスコットへ即時反映 |
| 表示スケール | スライダー | |
| 人格プロンプト | テキストエリア | 実体は `{userData}/prompts/persona.md`(04参照)。「リセット」で `persona.default.md` の内容に戻す。外部エディタでの編集も監視して反映 |
| 感情マッピング | テーブル編集 | 感情タグ→モデルの表情/モーション名。モデルから利用可能な表情・モーション一覧を取得して候補表示 |

### 4. 挙動設定

| 項目 | UI |
|---|---|
| ランダムトーク ON/OFF | トグル |
| ランダムトーク間隔(最小/最大 秒) | 数値入力 |
| 常に最前面 | トグル |
| OS起動時に自動起動 | トグル(`app.setLoginItemSettings`) |

## 設定反映の仕組み

- 設定変更は即座に `settings:set` でメインへ送り、electron-storeへ保存
- メインは変更内容に応じて各層へ反映(プロバイダ再生成、モデル再読込指示、タイマー再設定など)
- 「保存ボタン」方式ではなく**即時反映**とする(伺か系ツールの慣習に合わせる)

## トレイ常駐

- マスコットは `skipTaskbar` のためシステムトレイに常駐アイコンを置く
- トレイメニュー: 管理画面を開く / マスコット表示・非表示 / 終了
